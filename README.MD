### 디렉토리 설명
- **1차시 C__의 기본** : C++ 기초 문법 실습.
- **2차시 데이터의 변환** : 자료형 변환과 연산.
- **3차시 포인터의 사용** : 포인터와 구조체.
- **4차시 루프와 관계표현식** : 반복문과 관계 연산자.
- **5차시 분기 구분과 논리 연산자** : 조건문 및 논리 연산자 활용.

### 실행 방법
1. Terminal (cmd) 열기
2. 디렉토리 생성:
    ```bash
    mkdir "Git 파일"
3. 저장소 클론:
    ```bash
    git clone https://github.com/Moomin03/Standard_C.git
4. 디렉토리 이동:
    ```bash
    cd Standard_C
5. 컴파일 및 실행:
    ```bash
    g++ -o 저장할_파일 실행할_파일명.cpp
    ./저장할_파일
